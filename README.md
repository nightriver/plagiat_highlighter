# 🔍 Plagiat Highlighter

Веб-застосунок для автоматичного підсвічування текстових збігів і відмінностей у порівняльній таблиці плагіату у файлі MS Word (`.docx`).

## Що робить програма

**Вхід:** `.docx` файл з двоколонковою таблицею порівняння (дисертація ↔ оригінальне джерело).  
**Вихід:** той самий `.docx` з підсвічуванням:
- 🟡 **Жовтий** — збіг між текстами
- 🩵 **Бірюзовий** — відмінність

Усе форматування скидається до єдиного шрифту. Маркери сторінок, бібліографія та коментарі складача зберігаються без змін.

---

## Структура вхідного файлу

Файл повинен містити **одну таблицю**, 2 колонки, довільна кількість рядків.

| Ліва колонка | Права колонка |
|---|---|
| Маркер сторінки (`С. 35`) | Бібліографія |
| Текст дисертації | URL |
| *(порожній рядок)* | Маркер сторінки (`С. 88`) |
| Коментар складача | Текст оригіналу |

**Ліва колонка:** програма підсвічує текст між маркером сторінки і першим порожнім рядком. Маркер і коментарі після порожнього рядка залишаються без змін.

**Права колонка:** програма знаходить останній маркер сторінки (щоб не плутати з `С.` у бібліографії) і підсвічує текст після нього. Бібліографія і маркер залишаються без змін.

## Налаштування

| Параметр | За замовчуванням | Опис |
|---|---|---|
| Поріг схожості | 75% | Мінімальна схожість для fuzzy-збігу. 75% оптимально для морфології укр. мови (`суд` / `суду` = збіг) |
| Шрифт | **Calibri** | Також доступні: Times New Roman, Arial |
| Розмір шрифту | **14 pt** | Діапазон: 8–16 pt |

> Для слів ≤ 4 символи поріг автоматично знижується до 70%, щоб коректно зіставляти короткі відмінювані форми.

## Запуск локально

```bash
# 1. Клонувати репозиторій
git clone https://github.com/nightriver/plagiat_highlighter.git
cd plagiat_highlighter

# 2. Встановити залежності
pip install -r requirements.txt

# 3. Запустити
streamlit run app.py
```

Відкрийте браузер за адресою `http://localhost:8501`.

## Як працює алгоритм

1. **Анотація ячейок** — кожен параграф отримує мітку `plain` (зберегти без змін) або `text` (порівнювати і підсвічувати).
2. **Токенізація** — текст розбивається на токени (слова + пробіли). Переноси (`Ін- тернет` → `Інтернет`) нормалізуються.
3. **Вирівнювання** — `SequenceMatcher` зіставляє слова, `rapidfuzz` обробляє нечіткі збіги. Пробіли між двома словами з однаковим статусом отримують той самий статус (суцільне підсвічування).
4. **Запис** — ячейка перезаписується повністю: `plain`-параграфи як є, `text`-параграфи з підсвічуванням.
